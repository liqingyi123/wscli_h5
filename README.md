## Project setup
```
npm i 或 npm install
```

### 依赖项
```
compression-webpack-plugin@6.1.1  --save-dev
axios qs vant vue-cookies weixin-js-sdk -S
```

### 热更新的开发模式
```
npm run debug
```

### 编译压缩成生产测试环境资源包
```
npm run test
```

### 编译压缩成生产线上环境资源包
```
npm run online
```
